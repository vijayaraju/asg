Upgrade

The upgrade to the last version is ufficially supported just starting from the following versions:

  • 2.1.2
  • 2.1.1
  • 2.1
  • 2.0.1

2.1.2

  • Backup the config file located at /myasg/includes/inc_config.asp.
  • Overwrite all files in the folders /myasg and /myasg/includes with the new ones from the downloaded package.
  • Restore the config file.
  • Warning: Delete setup.asp and update.asp files from your server.

2.1.1

  • Overwrite all files in the folder /myasg with the new ones from the downloaded package.
    No need to overwrite the other /myasg subfolders.
  • Warning: Delete setup.asp and update.asp files from your server.

2.1

  • Overwrite the following files with the new ones
    • \myasg\config.asp
    • \myasg\config_common.asp
    • \myasg\count.asp
    • \myasg\pages.asp
    • \myasg\statistiche.asp
    • \myasg\stats_calendar.asp
    • \myasg\stats_daily.asp
    • \myasg\stats_hourly.asp
    • \myasg\update.asp
    • \myasg\stats.gif
    • \myasg\images\browser.asp
    • \myasg\images\engine.asp
    • \myasg\images\os.asp
    • \myasg\database\ip-to-country.mdb
  • Delete setup.asp and update.asp files from the server

2.0.1

  • Overwrite all old files with the new ones
  • Run update.asp file and follow the instructions to upgrade the database
  • Delete setup.asp and update.asp files from the server

To upgrade from other versions please check and follow the information posted on the forum.

On the forum you'll find a new section called Online Version containing bug fixes and program news.
We suggest you to often visit the forum to keep the application updated!

ATTENTION:
before update the application we strongly recommend to create a full backup of the database and old files.
