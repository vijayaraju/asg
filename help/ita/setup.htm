Installazione

L'installazione necessita unicamente di qualche piccola procedura manuale, mentre la maggior parte delle configurazioni è ora affidata ad un file di installazione.

PASSO 1.

Scompattate l'archivio .zip scaricato dal sito. Una volta ultimata la decompressione troverete la seguente struttura di file:

	Unzipped Folder
	|
	|-- myasg (contiene l'applicazione ed i file necessari da uploadare sul server)
	|-- help (contiene la documentazione di aiuto)
	|
	|-- readme_[language].htm (file di avvio della documentazione nella lingua specificata)

E' consigliato mantenere questa struttura di file e cartelle evitando rinominazione o potreste avere dei problemi nell'installazione.

PASSO 1b.

Nel caso venga rinominata la cartella 'myasg' sarà necessario variare il percorso anche nel file stats_js.asp

PASSO 2.

Adattate i percorsi di connessione al database ed i file con permessi di scrittura per la gestione delle skin.

ATTENZIONE:
Per il corretto uso dell'applicazione è necessario che il database ed i file specificati di seguito si trovino in una cartella con attivati i permessi di scrittura e nello specifico caso del database con disattivato il permesso di navigazione e download da browser per motivi di sicurezza.
Generalmente ogni hosting a pagamento provvede a fornire una o più cartelle con permessi specifici attivati direttamente dalla consolle di amministrazione del server.
Prima di procedere informatevi presso il vostro fornitore quali sono le cartelle con i permessi a vostra disposizione. Adattate le seguenti stringhe con il percorso al vostro database.

Nel dettaglio ecco l'elenco dei percorsi minimi da adattare.

File includes/inc_config.asp:
- Const strAsgPathFolderDb = "mdb/" 'Inserire il percorso alla cartella del database
- Const strAsgPathFolderWr = "mdb/" 'Inserire il percorso alla cartella con attivi i permessi di scrittura.
Può coincidere con quella del database.

File config.asp:
- Cambiare la riga di inclusione del file inc_skin_file.asp (<!--#include file="../database/inc_skin_file.asp") come specificato nelle informazioni, specificandone il percorso rispetto alla cartella specificata sopra nella variabile strAsgPathFolderWr.

PASSO 3.

Procedete con l'upload dell'intera cartella 'myasg' nella directory principale del server, ad esclusione del contenuto della cartella 'mdb'.

A upload completato, eseguite l'upload del contenuto della cartella 'mdb' come segue:
- File dei database nella cartella specifica al punto 1 (primo percorso)
- File inc_skin_file.asp nella cartella specifica al punto 1 (secondo percorso)

PASSO 4.

Avviate ora il vostro browser e digitate l'indirizzo http://www.mysite.com/myasg/setup.asp sostituendo l'indirizzo corretto al posto di www.mysite.com.

Se i punti precedenti sono stati eseguiti correttamente verrà aperto il procedimento di installazione dell'applicazione, in alternativa verificate l'errore restituito.
Molto probabilmente è relativo a un percorso errato al vostro database o al file di skin.

Procedete con il prossimo punto.
