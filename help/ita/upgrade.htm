Upgrade

L'upgrade all'ultima versione è ufficialmente supportato a partire dalle versioni seguenti:

  • 2.1.2
  • 2.1.1
  • 2.1
  • 2.0.1

2.1.2

  • Creare una copia di backup del file /myasg/includes/inc_config.asp.
  • Sovrascrivere tutti i file delle cartelle /myasg e /myasg/includes con quelli contenuti nel pacchetto scaricato dal sito.
  • Ripristinare il file salvato /myasg/includes/inc_config.asp.
  • Attenzione: Rimuovere i file setup.asp e update.asp dal server

2.1.1

  • Sovrascrivere tutti i file della cartella /myasg con quelli contenuti nel pacchetto scaricato dal sito.
  • Attenzione: Rimuovere i file setup.asp e update.asp dal server

2.1

  • Sovrascrivere i seguenti file con i nuovi
    • \myasg\config.asp
    • \myasg\config_common.asp
    • \myasg\count.asp
    • \myasg\pages.asp
    • \myasg\statistiche.asp
    • \myasg\stats_calendar.asp
    • \myasg\stats_daily.asp
    • \myasg\stats_hourly.asp
    • \myasg\update.asp
    • \myasg\stats.gif
    • \myasg\images\browser.asp
    • \myasg\images\engine.asp
    • \myasg\images\os.asp
    • \myasg\database\ip-to-country.mdb
  • Rimuovere i file setup.asp e update.asp dal server

2.0.1

  • Sovascrivere tutti i vecchi file con i nuovi
  • Avviare il file update.asp e seguire le istruzioni per aggiornare il database
  • Rimuovere i file setup.asp e update.asp dal server

Per upgrade precedenti seguire le informazioni riportate sul forum .

Sul forum troverete inoltre una nuova sezione chiamata Versione Online dove verranno inseriti gli ultimi aggiornamenti e le correzioni per eventuali bug rilasciati a supporto dell'ultima versione distribuita.
Suggeriamo quindi di visitarla spesso per avere un'applicazione sempre aggiornata e funzionale!

ATTENZIONE:
prima di eseguire un upgrade dell'applicazione si consiglia di eseguire una copia di backup completo dei file e del database.
