Configuration

Once installation is completed you need to configure the application.

Open the config file as written by the previous step and and fill out the form with the information required.
Confirm the information and in return, a message will confirm your application configuration.

Now you'll just need to put the monitoring string in each page of your site you would like to track.

WARNING:
When the configuration is completed delete setup.asp and update.asp files from your server or other users will be able to edit your settings.

For further informations please visit the web site or its forums.
