Primo Avvio e Configurazione

Una volta eseguita l'installazione potete procedere con la configurazione.

Avviate il file di setup come descritto al punto precedente e provvedete ad inserire le informazioni richieste.
Confermate i dati inseriti e se la procedura andrà a buon fine verrà mostrato un messaggio di conferma e l'applicazione sarà ora correttamente configurata.

Sarà ora sufficiente inserire la stringa di monitoraggio in ogni pagina del vostro sito che vorrete tracciare.

ATTENZIONE:
Cancellate dal server a procedura ultimata i file setup.asp e update.asp o altri utenti potrebbero modificare le impostazioni da voi inserite e/o causare danni al funzionamento dell'applicazione!

Per informazioni che non siano reperibili su questo Readme vi invitiamo a consultare il sito web ufficiale dell'applicazione ed il forum.
