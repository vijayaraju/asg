Site Activity Tracking

Once setup is completed just insert following script code between <body> and </body> tags in pages you would like to have monitored by the application.

<script type="text/javascript" language="JavaScript" src="http://www.mysite.com/myasg/stats_js.asp"> </script>

Replace www.mysite.com URL with the URL where the application is hosted.

If previous steps has been correctly followed, when you'll open a monitored page a little application icon will be displayed to confirm you the monitoring execution and data will be stored into the database.

To see the reports you can access from /programfolder/statistiche.asp or login from the page /programfolder/login.asp .
